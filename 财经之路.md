```

##  **一叶知秋** 

![](E:\夏伟\财经之路\图片\一叶知秋.jpg)

```
当股价猛烈上涨的时侯，巨大的赚钱效应刺激着人们的神经，让人们在兴奋中陷于疯狂，涨了还想涨，赚了还想赚，恨不得一夜暴富；人性的贪婪在股价上涨的刺激下，被放大到极致，同时主力机构正好利用这点达到顺利出货的目的；当散户在看着股价不断上涨，兴高彩烈的时侯，心里盘算着自己又赚了多少多少钱，然而却不知危险正悄悄向自己逼近。

股价在经过连续大幅上涨后，有了一定涨幅，在底部区哉介入的投资者账面有了盈利，此时有大量的获利盘，一旦有风吹草动，获利盘就会蜂拥而出，如果遇到利空消息或大盘走坏，卖盘就会互相残杀，股价就会出现急跌。

图2.8是上风高科的日K线走势图，股价由底部开始上涨，拉升到5元左右时，股价总体涨幅达100%，2009年2月16日股价顺势高开后，最高冲高到5.83元，在获利盘打压下，股价急速下跌，当日震幅接近9%。

随后股价又被主力买单迅速拉上来，同时吸引了大量散户跟风介入，当日收盘收了个带长下影线的阴线，当日震荡幅度较大，此时已表明该股筹码出现松动迹象，短线有调整要求；第二日股价低开低走，收出了一根大阴线，宣告上涨趋势暂时完结，股价出现了卖出信号。
```

##  **乌云盖顶** 

![](E:\夏伟\财经之路\图片\乌云盖顶.jpg)

```
股价经过前期大幅上涨后，并没有延续上涨走势，而是在相对高位出现大幅震荡走势，在这一阶段股价表现上涨乏力，每日成交量很大，在高位阴线渐渐增多，有如乌云袭来。

由于股价已有较大涨幅，在底部区哉介入的主力获利丰厚，但只是账面赢利，主力要想套现股票，必须在高位出掉手中的筹码；如果继续向上拉升股价，所用的操盘资金和交易成本也就相应扩大，同时由于股价过高，普通投资者有恐高心理，继续拉升股价风险较大。

此时主力通常利用股价反复震荡，来吸引短线客跟风介入，做出股价还要向上突破的假象，激发起人们的买入冲动情绪，在这一阶段主力出多进少，盘口如果接盘的人多，主力就出货，股价在主力大量卖盘的抛售下就会下跌，一旦接盘少了，主力再次利用大单迅速拉升股价，甚至创出新高来吸引买盘跟进，如此反反复复，散户买入股票后，没有赚钱是不甘心卖掉股票的，主力正好利用散户的这种心理，通过长时间震荡达到出货的目的。

投资者在实盘操作中，如果遇到这种走势形态，不要被股价的短期波动所诱惑，冲动介入做短线，这种走势由于股价涨幅较大，存在较大风险，即使股价还能涨，但上涨空间已不大，且股价随时有下跌的风险，盲目跟进的话，被套的概率较大。

假如高位介入后被套，股价跌破盘整的低点要及时止损，不要心存幻想等股价反弹保本出局，如果错过了最佳出局时机，往往越套越深。其中有一部分投资者在这个区域被套后，不是尽快止损出局，反而补仓，企图摊低成本，希望股价反弹自己能解套；如果大盘走好，股价有可能还会继续上涨，但一旦大势不好，股价将面临大幅下跌的风险。

在顶部区哉被套后补仓，这种思想是错误的，是致命的，股价从高位下跌，是由于主力和获利盘出货造成的，卖盘远远大于买盘，股价的走势是向下，补仓的话，只能是补仓的部分也被套，亏损进一步扩大，心存侥幸心理期望股价能反弹，将踏入万劫不复的深渊。
```

##  **高台跳水** 

![](E:\夏伟\财经之路\图片\高台跳水.jpg)

```
首先我们来看这种K线组合的表面形态：在下跌之前股价长时间一直处于小幅波动的盘整状态，突然某一天股价跌破盘整区域，拉出一个中阴线，并且成交量明显比前几日有所放大，在Ｋ线图上看，继续下跌的空间还很大，这种K线组合就是股价将要下跌的信号。

接着我们来分析在股价下跌前和下跌后持股者和场外投资者的心理状态：

股价下跌前长时间处于盘整状态，介入其中的投资者由于没有获利空间，大多采取一种观望态度，卖了吧，不赚钱是舍不卖的，万一刚卖出股价涨了怎么办；不卖吧光等着股价也不涨，左右为难，此时大多数持股者持有一种举棋不定的心理状态，等待股价的突破方向。

一旦股价没有向上涨，反而下跌了，股价的平衡状态被打破，此时就会引发大量抛盘卖出，导致股价出现急速下跌，如果持股的投资者在股价跌破盘整区间时，没有及时止损，则很容易被深套。

在场外观望的股民，看到股价下跌了，认为股价和原来相比便宜多了，可以抄底了，希望股价反弹自己能获利，于是在下跌途中介入了，但股价走势并没有如其所愿，底没有抄成，反而被套了，转而又加入杀跌的队伍。

在盘整区哉介入的投资者遇到这种破位急跌的股票，首先想到的是要及时止损，避免出现更大的损失；场外观望的投资者，在股价没有改变下跌趋势之前，不要急于抄底；应耐心等待大势转好后，选择走势形态向上的股票
```

##  **落日残阳** 

![](E:\夏伟\财经之路\图片\落日残阳.jpg)

```
股价整体走势，处于下跌趋势，下跌途中，偶尔一两天收阳线，表面看似股价有见底反弹迹像，但收阳以后，股价继续下跌，我们把这种下跌途中的阳线，称为落日残阳，这种走势形成的原因有两个：

一是，股价由于跌幅较大，短线抄底盘认为有利可以图，进场抄底，股价短时间内走出上涨行情。

二是，由于大势不好，股价急速下跌，介入该股的主力没有完全出掉手中的筹码，也就是主力被套，主力为了出掉手中的货，刻意拉升股价制造股价见底的假象，吸引抄底盘介入，好趁机出货。对于这种K线走势形态不要轻意抄底介入，避免股价下跌被套，股价真正见底要有一个漫长的过程。

有时跌的太多，股市就会出现反弹，一些急于扳本和抄底的散户就会进去抢反弹，但反弹过后，股价又会继续下跌，介入的投资者底没抄成反而被套了，2008年当股市跌到3000点左右时，人们认为股市见底了，纷纷进场抄底，有的投资者抵压房产，借代融资入市，但谁也没想到，股市在3000点没有站稳脚跟，一头裁倒，跌破3000点，直奔2000点而去，这时抄底的人们都蒙了，不知哪是底了，失望和绝望的心理恐慌性蔓延，大盘指数在人们恐慌性抛售的压力下，一直跌到1664点，才真正见底，当然这是后话，当时人们也不知这到底是不是底，都被跌怕了；当初在3000点左右介入的投资者损失惨重。

有的投资者会说了，大势不好的情况下，也有股票涨呀，只要有主力怕什么呀；这种认识是错误的，大部分股票的涨跌是随大盘波动的，大盘涨，大部分股票涨，大盘跌，大部分股票也跟着跌；如果主力逆势而为，吸了大肚子货，逆势拉升股价，则没有大量跟风盘买入，并且股价卖压较大，主力必然耗费大量资金托盘，即使利用资金实力将股价拉高，也是主力自拉自唱，在高位没有大量散户接盘，只能导致坐庄失败。

逆势拉升的股票随时有可能转为大幅下跌，投资者跟庄的话风险也是较大的。
```

##  **绝地反击** 

![](E:\夏伟\财经之路\图片\绝地反击.jpg)

```
股价前期一直处于震荡下跌走势，总体跌幅较大,属于严重超跌的类型，股价跌到了较低的区域，对于场外观望的资金有很大的吸引力；突然一两天股价大幅拉升，成交量伴随放大，后市股价不断创新高，就像被用力向下掷的皮球遇到地面突然强劲反弹一样，我们把这种K线形态组合称之为绝地反击。

这种走势一般是受利好因素导致股价大幅上涨，（也有前期被套主力，突遇利好拉升股价自救的，我们暂且不表），股价起动前，没有任何征兆，这种K线形态走出后，投资者要观注利好消息的实质性和公司基本面的变化情况，可根据当时的大势背景决定参与的仓位大小。

对于已大幅上涨的个股不要再盲目追高买入，以防股价调整被套，因为股价短时间大涨，在底部区域介入的投资者有获利出局的心理，并且前期套牢盘由于股价的大涨，接近自己的成本区，被套已久，好不容易股价涨了，如遇到大赦的猴子，拍拍屁股要赶快走人，大多数被套的投资者都是这个心理，如果没有大量的买盘承接这些卖盘的话，股价随时存在调整的可能性。

这种K线形态操作的前提条件是，股价经过前期大幅下跌，股价极低，公司基本面有转好迹象，有实质性利好消息，大势背景转好，股价处于刚起动的低位区域，并且有成交量配合。
```

##  **随波逐流** 

![](E:\夏伟\财经之路\图片\随波逐流.jpg)

```
股价在一个窄幅区间震荡波动，每日的涨跌幅度并不大，成交量也没有明显的规律性，总体走势呈现震荡盘整状态，股价走向没有明显的方向，看似如同随风波动的海面一样。

这种K线形态出现的空间位置有多种，一是下跌中的盘整状态，二是上涨中的盘整状态，三是主力在高位出货的盘整状态（主力在高位出货的走势我们在以后会讲到，这里暂且不表）；总之我们可以把没有明确的突破方向的K线运动形态统称为盘整状态，处于盘整状态的股票，由于没有明确的参考方向，所以投资者介入后成功概率较低，甚至会出现亏损。

那么已经介入其中的投资者怎么办？已经介入其中的投资者可以参照股价波动的箱底做为止损点，假如股价跌破了箱底，说明股价继续下跌的概率较大，你在里面呆着已没有多大意义，有可能被套，不如趁早点出来；做为场外的观望者，在股价没有明确的上拉动作，最好不要轻意介入处于盘整状态的个股，股票没有突破方向，你介入成功的概率是很低的，要耐心等待股价突破盘整区域，真正展开拉升动作后，再介入。

我们来详细分析下盘整状态个股形成的原因：首先说下下跌中的盘整，股价经过长时间下跌后，在高位被套的股民舍不得割肉，卖盘越来越少，卖压变小后，股价出现止跌迹象，但并不代表股价已见底，股价开始在一个相对平衡的状态波动，此时场外的观望者看到股价止跌了，开始入场，但总体买入的人并不多，卖出的人也不多，买卖双方都很谨慎，都在等待股价的突破方向；这种下跌中的盘整，如果遇到大势不好或个股基本面发生变化，股价跌破盘整区域，继续向下跌的概率还是较大，所以下跌中的盘整区域不要轻意重仓介入操作。

我们接着再来谈谈上涨中的盘整状态，上涨中的盘整状态前提条件是股价由底部开始上涨，并且已有一定涨幅，由于股价从底部上涨已有一定涨幅，有一部分在底部获利的投资者开始卖股票，股票的筹码出现松动迹象，此时买卖双方处于一个短暂的平衡状态；介入其中的主力资金同时会利用散户这种获利出局的心理，让股价在一个区间震荡波动，将持股不坚决的投资者震出局减轻拉抬的压力，让另一部看好该股的投资者入场，从而抬高散户的持股成本，以利以后的拉抬股价。

总之，遇到这种盘整状态的个股，首先要分清它是下跌趋势中的盘整还是上涨趋势中的盘整，同时参考大盘的走势好坏，股价所处的空间位置来决定如何操作。
```

##  **聚宝盆** 

![](E:\夏伟\财经之路\图片\聚宝盆.jpg)

```
股价经过前期大幅下跌后，上涨空间被打开，随着大盘走势逐渐转好，场外观望的主力资金开始进场，在图2.14中我们可以看到随着股价止跌企稳，成交量同时也伴随放大，说明有主力资金在吸货，主力有可能提前撑握个股的某些利好消息；股价回落时，成交量缩小，股价经过长时间的震荡在高位区域介入的投资者由于受不了折磨，纷纷割肉出局，而此时低位的大部分筹码则被主力资金收集；在这个区域筹码换手率达90%以上，说明大部分在高位被套的散户已割肉，主力撑握了大量的廉价筹码，只等待合适的时机拉升股价。

主力拉升股价前，大多会参考大盘的走势好坏，以减轻拉抬的阻力；随着股价不断上涨，成交量同时伴随放大，主力资金要把所有卖盘通吃才能拉高股价，所在在K线图上我们看到成交量放大，股价在主力资金的推动下，重心不断上移，在日线图上看阳K线明显比阴K线多，结合左边的K线走势来看在K线图上形成一个盆状底部形态，我们形象称之为聚宝盆。

这种K线形态走出来后，股价继续创新高的概率较大，随着股价不断上涨，在股价突破前期套牢区域时，遇到了割肉盘，股价展开了调整来消化套牢盘和浮动筹码，卖盘被消化后，股价继续上攻突破盘整区域，在股价突破盆沿的区域时，对于投资者来说，是个最佳的介入位置，此时介入盈利概率较大；假如股价突破盆沿的区域后，股价没涨，反而下跌了，此时要注意股价走势不正常，主力有可能感觉到卖压较大，需要洗盘，作为投资者在情况不明的情况下，应先退出观望。

操作这种股票的前提条件是，大势转好，目标股的基本面了解，股价处于底部区域，有明显的见底特征，在K线图上可以明显看出主力资金介入的痕迹。

##  **超跌反弹V型底：**

俗称“尖底”，型态走势象V字型。股价经过长时间大幅下跌后，卖盘越来越少，在顶部区域套牢的投资者由于损失较大，不肯割肉，随着卖盘的减少，抛压越来越小；随着股价不断下跌，股价跌得已相当低，具有极大的吸引力，已属严重超跌，场外观望的投资者抄底意愿相当强烈，总想趁最便宜的时侯买入股票；如果大势转好，则会有大量资金介入抄底。

主力一般选择在大势不好的情况下进场吸筹，个股一旦有主力资金介入，多空力量必然会发生变化，股价会在主力的主动性买盘的推动下，不知不觉慢慢走高。主力建仓一般是悄悄进行的，有计划地控制股价在一个价格区域内波动，当股价经过一段缓慢拉升之后，主力通常会以少量筹码将股价再次打压下来，迫使持股不坚定的投资者卖出手中的股票，同时主力可以在较低的价格继续建仓，如此反复，直到主力吸够筹码为止，在K线图上就形成底部K线形态。

在股票实战操作中，经常出现的五种经典底部K线形态有V形底、双肩底、头肩底、圆弧底和多重底。

##  **潜龙出海** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005346.jpg)

```
股价经过前期大幅下跌后，长时间在低位一个狭窄区间震荡波动，每日的成交量很小，表面看似没有人理会，其实是主力利用长时间小幅震荡来达到悄悄吸筹的目的；随着时间的推移，股价的波动幅度越来越小，股价重心开始上移，说明筹码大部分已集中到主力手中，主力择机选择拉升时机，主力一般选择在大势走好的情况下，人气旺盛，跟风盘较多拉抬股价，可以有效节省操盘成本，同时接盘的人多，可以在高位成功出货。

在图2.1中可以看到，随着主力吸筹任务完成，突然一两天成交量放大，股价创出新高，股价重心不断上移，呈现放量上攻动作，形态上看好似一条潜伏海底的蛟龙，腾空跃出海面。

在股价跃出海面的区域是投资者最佳的介入位置，此时股价是由低位向上拉升，有主力资金拉抬股价，且时机成熟，此时介入风险极小，且介入后盈利的概率很大，这种K线形态走出来后，后市继续创新高的概率较大，投资者在实战操作中，要对这种典型的走势形态加以关注。

有些投资者认为股价已从底部涨了好大一段，不敢介入了，这种思想认识是错误的，反而失去了最佳的介入时机，此时股价突破盘整区域时，只不过是洗盘结束后刚进入拉升段，进入拉升段的股票，往往是最赚钱的一段。

##  **拨云见日** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005439.jpg)

```
上涨途中的调整K线组合形态，股价由底部开始向上拉升，在上攻途中突然遇到大量卖盘的打压，上攻受阻，随后股价展开了连续几日的阴线调整，股价似无力上攻，成交量也伴随股价回调缩量；随着卖盘的越来越少，另一部分看好该股的投资者开始买入股票，受大量买盘的介入，股价重拾升势，突破盘整区域，创出新高，如乌云散去艳阳高照。

这种K线组合形成的原因有二：

其一,股价由底部开始上涨，在底部区域介入的投资者随着股价的不断上涨大部分已获利，获利盘越来越多，投资者的卖出心理随着股价的不断上涨而增大，一部分投资者开始卖出股票，股价随着卖盘的增大，不断下跌；在场外观望的投资者看到股价回调有机会介入，看好该股的投资者则趁股价调整的时机介入，随着获利盘的回吐压力减小，股价在新的买盘资金的推动下，继续上涨。

其二，当股价上涨一段时间后，股价出现调整，在底部介入的投资者由于不知是什么原因股价不涨了，由于害怕到手的赢利再失去，往往就会卖出股票；散户的这种心理，正好被主力利用，主力经常利用这种K线形态走势来达到洗盘的目的，主力在拉升股价的过程中，由于堆移了大量的获利盘，为了减轻以后的拉抬压力同时让另一部分散户入场，以提高散户的总体持股成本，此时主力通常会让股价在一个区间震荡波动，以达到清理浮筹的目的，待浮筹被有效清理后，主力再次拉升股价。

这种K线形态组合变数较多，如果大势不好，或突遇公司基本面变化，这种形态通常转化为主力出货的顶部形态；在实盘操作中，投资者应慎重对待这种走势形态，如果股价涨幅过高，主力有足够的出货空间，没有买入的投资者不要盲目跟风追涨。
```

##  **芝麻开花** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005534.jpg)

芝麻开花节节高，在K线图中我们可以形象得把它比做连续涨停的K线形态。

股价启动前，每日波动幅度较小，成交量处于缩量状态，主力通过这种不温不火的走势，消磨投资者的意志，来达到洗盘的目的，在浮筹被有效清理后，时机成熟，主力开始大幅拉升股价，拉升阶段几乎是一气呵成，股价连续涨停，中间没有像样调整。

这种连续拉升的股票一般出现在大势背景良好，主力吸筹充分，时机成熟；对于这种股票，投资者如果错过了股价起动初期的介入时机，不应被股价大幅上涨而激发的亢奋心理所诱导而盲目介入，股价涨的越高，介入的风险越大；股价一旦不再涨停，说明股价短期内将见顶。

当时大盘开始走好，该股又具备新能源炒作概念，新能源行业受到国家政策支持，主力借机大幅拉升该股；当股价大幅上涨时，散户受贪婪心理的影响开始追高买入股票；主力经常利用某个概念来激发投资者对股价的想像空间，实现在高位卖出股票的目的，当人们兴高彩烈的因为买到大幅上涨的股票而高兴时，没有想到反而掉入了主力设置的圈套当中，人们之所以乐此不疲参与股市交易，正是因为股市的魅力不在于股票所基于的价值，而在于股票给投资者提供的幻想。

当一个概念被股民抛弃后，主力机构又会提出新的概念，比如网络股，高新技术，新能源，有色金属等概念来激发起股民的信心，也正是由于这此层出不穷的概念，股市里才有了此起彼伏的行情，才有了人们投资股市的机会，这也是股市顽强的生命力之所在。

##  **上攻受阻** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005616.jpg)

```
上涨途中突遇放量大阴线，股价从底部开始上攻，由于短期涨幅较大，开盘后大量卖盘抛售，买方无力承接大量的抛盘，股价只能节节败退；收盘时收出一根带量大阴线，但上涨趋势并没有改变，后市走势要所变化；这种K线组合多出现在上涨中的调整，也有的主力会利用这种走势出货，杀伤力极大。

这种K线组合形成的原因有二：

其一，股价短时间内涨幅较大，存在较多的获利盘，股价短时间内大幅上涨，获利盘蜂拥而出，造成股价急跌。

其二，主力利用这种走势洗盘，达到清理获利盘和解套盘的目的，以便在以后的拉升过程中减轻压力，节省拉抬成本，同时主力利用高抛低吸降低持股成本；这种K线组合形态，走出来后，后市变数较大，如果操盘主力感觉抛压较重，后市可能继续洗盘，如果主力感觉浮筹被有效清理，第二天稍微调整后，会再次上攻。

普通投资者遇到这种走势，是最头疼的，因为你不知主力洗盘幅度的大小，如果持股不卖，股价继续下跌怎么办？如果卖了吧，股价调整后，要是涨了怎么办？遇到这种K线组合形态，持股者可以根据自己的介入成本设立持股位，在确保资金安全的前提下最大化盈利，如果你看不明白是不是主力的洗盘行为，最好先退出观望；如果你能看懂就是主力在洗盘，要坚定自己的信心，忍受住洗盘的折磨，持股待涨；同时参考成交量的变化，以及大盘走势，进一步确认自己的分析。

有些个股，形态走势非常好，第二天却大幅下跌，大盘突然走坏或个股受某方面消息影响导致股价突然下跌，比如超过涨幅限制突然停盘的个股是投资者最忌讳的，股票遭到停牌后，再开盘通常走势会大如前，甚至有些个股从此就萎靡不振。

主力也经常利用此种走势在高位抛出在低位回补筹码来达到获利的目的，主力可以在高位卖出股票，然后让股价下跌后，再回补筹码，赚取差价，这也是主力在操盘过程中利润来源的一部分；如果投资者见股价下跌了就止损，止损后没多久股价却又回到了原来的价位，这样反复止损，也会造成资金的巨大亏损，投资者在实盘操作中应根据大盘走势的好坏及个股价位的高低采取相应操作，如果盘口观察主力处于拉升初期，股价突然下跌，并且成交量缩小，可以判断主力借机回调的概率较大，即使短线被套，也大可不必惊慌，可持股等待股价回升；当然如果大盘和个股处于高位顶部区哉则另当别论。

##  **回马枪** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005646.jpg)

```
股价前期一直处于震荡盘升走势，突然股价遭遇大量卖盘打压，快速下跌，成交量没有明显的规律性，连续几根阴线将盘升形态打破，然而随后的几天里，股价却止跌回升，又迅速收复失地，创出新高，并且带量突破前期高点，这种形态组合大多是由于主力洗盘行为所致，也有股价被停盘后，突然开盘大量卖盘出逃所致，我们将这种形态组合称为回马枪。

这种形态的变化较多，杀伤力大，下跌前没有任何征兆，股价本来走的好好的，突然下跌，在下跌前介入的投资者大多被套，被套后因为害怕股价继续下跌而及时止损卖出股票，没想到止损后没几天股价又回到了止损前的价位，此时出局的投资者害怕股价再次回落通常不敢再介入该股，该股反而大幅拉升不断创出新高，止损卖出的投资者因为害怕再次被套而踏空行情；主力也正是因为利用了散户的恐惧心理先让股价大幅下跌诱空，逼迫散户止损卖出股票，当散户卖出后，主力再大幅拉升股价，当散户发现自己被骗了，该股是一只大黑马时，此时股价已高高在上了，如果散户心存报复的心理再次介入该股，通常又会中了主力的圈套；由于主力操盘手的操盘手法不同，投资者遇到这种K线组合形态还要辨证分析，结合当时大盘走势，个股基本面情况，股价所处的空间位置来综合分析。

一般情况下，如果股价处于从底部区域刚进入拉升段后出现这种K线形态，随后股价又收复失地，再次拉升突破前期高点时，是投资者介入的良机，通过分析我们可以判定股价的回调是主力的洗盘行为，此时介入盈利的概率较大；如果股价涨幅已大，处于高价区，投资者应慎重对待，不应在高位盲目介入，以防落入主力设置的圈套。
```

##  **假痴不颠** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005722.jpg)

```
宁伪作不知不为，不伪作假知妄为。静不露机，云雪屯也。

假痴不颠是三十六计中的第二十七计，此计运用此象理，是说在军事上，有时为了以退求进，必得假痴不癫，老成持重，以达后发制人。这就如同云势压住雷动，且不露机巧一样，最后一旦爆发攻击，便出奇不意而获。

假痴不癫，重点在一个“假”字。这里的“假”，意思是伪装，装聋作哑，痴痴呆呆，而内心里却特别清醒。

在图2.6中我们可以看到，11月26日一根放量中阴线，随后两天股价却连续涨停，主力正是利用了假痴不颠这一计，故意制造一根放量中阴线来迷惑普通投资者，使散户以为股价短期内可能回调，或者股价要下跌了，在底部介入的投资者和前期被套的投资者害怕股价下跌则会卖出股票，一旦散户卖出手中的股票，主力则强劲拉升股价，让卖出股票的投资者踏空行情，迫使想买股票的投资者在高位追涨接盘。

主力在操盘过程中经常利用这种走势，表面上装疯卖傻，给人以碌碌无为的印象，实则隐藏自己的意图，掩盖内心真正拉升股价的目的，以免引起散户大量跟风，等待时机，实现自己的拉抬目的；当主力再次拉升股价，暴露出真正拉升股价的目的后，投资者可及时跟进，通常会获得不菲的收益。
```

##  **更上层楼** 

![](E:\夏伟\财经之路\图片\屏幕截图 2022-07-28 005802.jpg)

```
一只股票的升幅，在一定程度上是由主力介入的资金量大小决定的，主力动用的资金量越多，持有的筹码越多，相对来说股票的涨幅越大，主力为了能顺利出货，必须将股票拉升出大幅的上涨空间。

当主力机构建仓成功后，依据大势的好坏，决定拉升时机，如果大势走势良好，主力会在很短的时间内放量拉升股价，此时主力大笔买入，扫荡所有卖盘，使股价迅速脱离主力持仓成本区，此时持股的散户寥寥无几，跟风的散户则看到股价大幅上涨，追涨买入，持股成本比主力要高许多，主力便有了操作的主动权。

当股价拉升到一定价位时，主力则借机洗盘，打破一些重要的持术位，让散户以为股价见顶了，恐慌性卖出股票；当跟风盘被清理掉后，主力再次大幅拉升股价，同时散布利好消息吸引人气，这一阶段股价往往以连续涨停收盘，激发散户的购买欲望；当大量散户被股价的大幅上涨所引诱而纷纷跟进时，此时主力正好利用人们失去理智的买入下，大量出货。

散户获取信息的渠道不畅通，没有办法撑握上市公司的和主力机构的真实情况，本身就处于劣势，很容易被主力机构所利用，不是在高位追高接盘，就是在低位割肉出局；当一支股票跌的时侯，他们不知道为什么跌，因而非常恐惧，结果往往在很低的价位割肉出局，主力机构正好利用这一点来达到自己的目的。

图2.7是万向钱潮的K线走势图，股价在低位经过长时间震荡盘整后，随着大盘走势转好，股价开始带量上涨，股价由3.2元开始启动，经过连续几日上涨后，涨到4.5元时，股价在这一价位区间展开了数日的调整，每日震荡幅度较小，成交量较前几日明显萎缩。

主力此时让股价在一小幅区间震荡，意在让在低位介入的获利盘出局，场外的投资者进场，新资金的进场，可以有效提高市普通投资者的持股成本；股价经过几日强势调整后，再次放量拉升，在股价盘整时卖出的投资者发现被主力骗了，错失了后面的大段利润，如果内心耿耿于怀，存有报复的心理，假如再次介入的话，有可能落入主力的陷阱，接手最后一棒；股价再次大幅拉升，往往进入了主力的出货阶段，此时受不了股价暴涨诱惑的投资者，如同飞蛾扑火。
```
